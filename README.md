# DIXON — сайт сервисного центра

Современный редизайн сайта [dixon-sc.ru](https://dixon-sc.ru/). Все разделы и цены сохранены. Таблица цен сделана удобнее: сначала выбирается серия телефона (A, M, Note, S, Tab, Z Flip, Z Fold), затем показываются модели и цены на дисплей, аккумулятор и заднюю крышку.

## Как запустить локально

**Вариант 1 — без установки (самый простой)**  
Просто откройте файл `index.html` в браузере (двойной клик по файлу или «Открыть с помощью» → браузер). Сайт будет работать.

**Вариант 2 — с локальным сервером (удобно для разработки)**  
Если хотите запустить как сайт по адресу `http://localhost:...`:

1. Установите **Node.js** с [nodejs.org](https://nodejs.org/) (версия LTS).
2. В папке проекта откройте терминал и выполните:
   ```bash
   npx serve .
   ```
   Или установите один раз и потом запускайте:
   ```bash
   npm install -g serve
   serve .
   ```
3. Откройте в браузере адрес, который покажет команда (обычно `http://localhost:3000`).

**Вариант 3 — через расширение в VS Code / Cursor**  
Установите расширение «Live Server» и нажмите «Go Live» — страница откроется с автообновлением при сохранении файлов.

## Логотип

Чтобы на сайте отображался логотип компании:

1. Сохраните логотип с текущего сайта или из макета.
2. Положите файл в папку `images/` и назовите его **`logo.png`** (или `logo.svg`).
3. Если используете SVG, в `index.html` замените `images/logo.png` на `images/logo.svg` в двух местах (в шапке и в подвале).

Если файла `images/logo.png` нет, в шапке и подвале будет показываться текст «DIXON».

## Структура проекта

- `index.html` — разметка страницы
- `styles.css` — стили и анимации
- `app.js` — табы серий, таблица цен, бургер-меню, скрытие шапки при скролле
- `data/prices.js` — данные по ценам (все модели и серии)
- `images/logo.png` — сюда положить логотип

Цены и контакты можно править в `index.html` и в `data/prices.js`.

## Скрытое управление ценами (админка)

Страница **`admin.html`** — вход по логину и паролю, без базы данных. Цены сохраняются в браузере (localStorage) и подставляются на главной вместо данных из `data/prices.js`.

1. Открой в браузере: **`https://ваш-сайт.ru/admin.html`** (или локально: `http://localhost:3000/admin.html`).
2. Логин и пароль по умолчанию заданы в `admin.js`: логин `dixon`, пароль `dixon2024`.
3. **Обязательно смени** логин и пароль в `admin.js` (переменные `ADMIN_LOGIN` и `ADMIN_PASS`).
4. После входа можно менять цены по сериям и нажать «Сохранить» — они сохранятся в localStorage. «Сбросить к исходным» удаляет сохранённые цены. После сохранения обнови главную страницу (F5), чтобы увидеть изменения.
